### Prerequisites

- PHP >= 7.0
- WordPress >= 4.6
- WooCommerce >= 3.0
- Node.js (for frontend development)

### Installation

1. Clone the repository
2. Install PHP dependencies:
```bash
composer install
```

3. Install Node.js dependencies:
```bash
pnpm install
```

4. Build assets for production:
```bash
bash clean.sh
```
